# Data Engineering in 2026 and Beyond

> Agentic coding means not asking another team to fix our problem

Before this year, data engineers at Meta have focused on the artifacts created downstream of client applications. While we may use server processing to enrich data, we almost never work on the iOS or Android code that actually generates that data. We relied on Software Engineers (SWEs) to implement new features and correctly define its associated logging.
